# MathForge

중학교부터 고등학교까지 수학문제를 생성하고 보관하는 웹앱 프로토타입입니다.

## 실행 방법

별도 설치 없이 `index.html`을 브라우저에서 열면 됩니다.

```text
C:\CursorWork\myProject\mathWork\index.html
```

로컬 서버로 실행할 때는 3100번 포트를 사용합니다.

```text
http://127.0.0.1:3100/index.html
```

Windows에서 더블클릭 실행하려면 `start-mathforge.bat`을 사용하세요.

다른 PC, 웹사이트, 데스크탑 앱 배포 방법은 `DEPLOY.md`를 참고하세요.

## 포함 기능

- 대시보드: 총 생성 문제, 객관식 비율, 평균 난이도, 최근 레벨 표시
- 문제 생성: 중학교/고등학교, 학년, 단원, 문제 유형, 난이도, 생성 개수 선택
- 문제 유형: 객관식, 주관식, 서술형, 참/거짓, 혼합
- 생성 개수: 한 번에 1~10개
- 문제 보관함: 브라우저 `localStorage`에 자동 저장
- PDF 내보내기: 브라우저 인쇄창에서 PDF로 저장
- 엑셀 내보내기: 현재 생성 결과, 세트, 전체 보관함을 `.xls` 파일로 저장
- 문제지만 출력, 정답지만 출력
- 메일 보내기: 생성 결과를 기본 메일 앱으로 전달
- 생성 세트 보관: 문제 생성 1회 단위로 보관함에서 구분
- 사용자 가이드와 배포 안내 메뉴
- 설정 메뉴: Gemini, OpenAI, 로컬 LLM API Key/모델/엔드포인트 저장, 수정, 삭제, 연결 테스트
- 파일 저장/불러오기: 전체 기록과 설정을 JSON 파일로 백업/복원

## AI API 선택지

### 1. Google Gemini API

사용자가 말한 "구글 무료 API" 후보로 가장 적합합니다. 무료 티어로 프로토타입을 시작하기 좋고, 문제 생성과 풀이 설명 생성에 충분히 활용할 수 있습니다.

권장 구조:

```text
브라우저 → 백엔드 API → Gemini API
```

API 키를 브라우저에 직접 넣으면 노출되므로, 실제 서비스에서는 반드시 백엔드에서 호출하세요.

현재 프로토타입의 `설정` 메뉴는 비교와 화면 흐름 확인을 위한 기능입니다. 입력한 값은 브라우저 `localStorage`에 저장됩니다.

설정 메뉴에서 제공자별 모델을 선택할 수 있고, `직접 입력`을 고르면 목록에 없는 모델명도 사용할 수 있습니다. `연결 테스트`는 현재 입력된 API Key, 모델, 엔드포인트로 간단한 요청을 보냅니다. 브라우저에서 직접 호출하기 때문에 일부 API는 CORS 정책으로 실패할 수 있습니다.

### 2. OpenAI API

수학 문제 조건 준수, 정답/해설 형식 고정, JSON 출력 안정성이 중요하면 좋은 선택입니다. 비용은 사용량 기반으로 관리해야 합니다.

### 3. 로컬 LLM

Ollama 같은 도구로 로컬 모델을 붙일 수 있습니다. 비용과 개인정보 통제에 유리하지만, 서버 성능과 모델 품질 관리가 필요합니다.

### 4. 하이브리드 방식

초기에는 Gemini 무료 티어로 시작하고, 품질이 중요한 기능만 OpenAI API를 쓰는 방식도 현실적입니다. 예를 들어 문제 초안은 Gemini, 정답 검증은 OpenAI 또는 별도 수학 검증 로직으로 나눌 수 있습니다.

## 배포 방향

### 웹사이트

- 초기: GitHub Pages, Netlify, Vercel 같은 정적 호스팅
- AI API 연동 이후: Next.js, Express, FastAPI 중 하나로 백엔드 추가

### 데스크탑 앱

- Tauri: 가볍고 배포 파일이 작음
- Electron: 자료와 생태계가 크고 구현이 쉬움

### 모바일 앱

- PWA: 웹앱을 설치형처럼 제공할 수 있어 초기 비용이 낮음
- React Native 또는 Flutter: 앱스토어 배포와 네이티브 기능이 중요할 때 적합

## 복사, 내보내기, 메일

- `복사`: 마지막 생성 결과 또는 보관함의 특정 세트를 클립보드에 복사합니다. 브라우저 Clipboard API가 막히면 fallback 복사를 시도합니다.
- `PDF`: 인쇄 미리보기를 열고 PDF로 저장할 수 있게 합니다.
- `엑셀`: `.xls` 파일로 저장합니다.
- `문제지만 출력`: 정답과 풀이 없이 문제지만 인쇄합니다.
- `정답지만 출력`: 정답과 풀이만 인쇄합니다.
- `메일 보내기`: `mailto:` 링크로 기본 메일 앱을 엽니다. 본문이 너무 길면 메일 앱 대신 내용을 복사해 붙여넣을 수 있게 합니다.

## JSON 백업/복원

설정 메뉴의 `파일 저장/불러오기`에서 전체 기록을 JSON으로 저장하거나 복원할 수 있습니다. 백업 파일에는 API Key가 포함될 수 있으므로 외부 공유에 주의하세요.
