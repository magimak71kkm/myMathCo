사용자 가이드

빠른 시작

  1. 문제 생성학교급, 학년, 단원, 유형, 난이도, 개수를 선택합니다.
  2. 생성 결과 확인정답과 풀이를 검토하고 필요한 경우 다시 생성합니다.
  3. 보관함 확인생성한 문제는 자동으로 생성 세트 단위로 저장됩니다.
  4. 출력 또는 백업PDF, 엑셀, 문제지, 정답지, JSON 백업 중 필요한 방식을 선택합니다.

문제 생성 조건

  • 학교급은 중학교와 고등학교 중 선택합니다.
  • 학년을 바꾸면 선택 가능한 단원이 자동으로 바뀝니다.
  • 문제 유형은 객관식, 주관식, 서술형, 참/거짓, 혼합을 지원합니다.
  • 수업 용도, 학생 수준, 풀이 수준을 지정하면 강사용 분석이 함께 생성됩니다.
  • 한 번에 1개부터 30개까지 생성할 수 있습니다.

강사용 분석

  • 각 문항에는 예상 풀이 시간, 핵심 역량 태그, 수업 포인트가 표시됩니다.
  • 흔한 오답 포인트를 켜면 학생들이 자주 놓치는 조건을 함께 확인할 수 있습니다.
  • 채점 기준을 켜면 서술형 첨삭과 부분 점수 부여에 활용할 수 있습니다.
  • 도형/그래프 우선을 켜면 시각 자료가 필요한 단원에서 그림 문항 생성을 강화합니다.

보관함 사용

  • 문제는 생성할 때마다 하나의 세트로 저장됩니다.
  • 세트 제목 영역을 누르면 문제 세부정보를 펼치거나 접을 수 있습니다.
  • 세트별로 복사, PDF, 엑셀, 문제지 출력, 정답지 출력, 메일 보내기를 사용할 수 있습니다.
  • 127.0.0.1과 localhost는 저장소가 다르므로 한 주소를 계속 사용하는 것이 좋습니다.

출력과 내보내기

  • PDF는 인쇄 창에서 PDF로 저장을 선택하는 방식입니다.
  • 엑셀은 .xlsx 파일로 내려받아 스프레드시트에서 열 수 있습니다.
  • 문제지만 출력은 정답과 풀이를 숨깁니다.
  • 정답지만 출력은 정답과 풀이 위주로 정리합니다.

API 설정

  • 설정 메뉴에서 Gemini, OpenAI, 로컬 LLM 정보를 관리합니다.
  • 모델은 목록에서 선택하거나 직접 입력할 수 있습니다.
  • 저장한 AI 제공자와 모델은 문제 생성 화면의 기본값으로 자동 적용됩니다.
  • 글씨체는 설정 메뉴에서 선택하며 화면, 인쇄, PDF 출력에 함께 적용됩니다.
  • 연결 테스트는 현재 입력된 키, 모델, 엔드포인트로 간단한 요청을 보냅니다.
  • 실제 배포 서비스에서는 API Key를 브라우저가 아니라 백엔드에 저장해야 합니다.

저장과 백업

  • 기본 저장은 브라우저 localStorage에 자동으로 이루어집니다.
  • 설정 메뉴의 JSON 백업으로 전체 기록을 파일로 저장할 수 있습니다.
  • JSON 복원으로 다른 브라우저나 다른 주소에서도 기록을 가져올 수 있습니다.
  • 백업 파일에는 API Key가 포함될 수 있으므로 공유에 주의하세요.

문제 검수 팁

  • 수업 전에 정답과 풀이를 반드시 확인하세요.
  • 객관식은 오답 선지가 너무 obvious하지 않은지 살펴보세요.
  • 서술형은 채점 기준을 따로 보완하면 활용도가 높아집니다.
  • 같은 조건으로 여러 번 생성해 난이도와 표현을 비교해보세요.

문제가 생겼을 때

저장한 문제가 안 보일 때
주소가 localhost인지 127.0.0.1인지 확인하세요. 두 주소는 브라우저 저장소가 분리됩니다.

PDF가 바로 저장되지 않을 때
인쇄 창에서 프린터 대상을 PDF 저장으로 바꾸면 됩니다.

API 테스트가 실패할 때
키 권한, 모델명, 엔드포인트, 브라우저 CORS 정책을 확인하세요.

Deployment

배포 및 AI API 안내

웹사이트 배포

초기 버전은 정적 사이트로 Netlify, Vercel, GitHub Pages에 바로 올릴 수 있습니다. 서버 API가 필요해지면 Next.js 또는 Express 백엔드를 붙이는 방식이 좋습니다.

데스크탑 앱

웹앱을 Electron 또는 Tauri로 감싸면 Windows/macOS 데스크탑 앱으로 배포할 수 있습니다. 가볍게 가려면 Tauri가 유리합니다.

모바일 앱

React Native, Flutter, 또는 PWA 방식을 고려할 수 있습니다. 우선 PWA로 시작하면 설치형 앱처럼 배포하면서 개발 비용을 줄일 수 있습니다.

AI API 선택

Google Gemini API는 무료 티어 테스트에 좋습니다. 품질과 안정성이 더 중요하면 OpenAI API, 내부망이나 개인정보 보호가 중요하면 로컬 LLM을 검토하세요.
